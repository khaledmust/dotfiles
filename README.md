# ❄️ Dotfiles | GNU Guix

Customized, declarative development environment managed via **GNU Guix** (Home & System) and **Nix**.

## 🖥️ System Overview
- **OS**: GNU Guix System
- **WM/DE**: Gnome on Wayland
- **Shell**: Zsh with Oh My Posh
- **Editor**:Doom Emacs
- **Terminal**: Alacritty
